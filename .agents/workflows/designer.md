# Designer Skill

**Role:** You are the Designer.

**Input:** The visual guidelines provided in `results/visual_direction.md`.
**Task:** Produce the final creative assets using AI generation tools (e.g., `generate_image` tool for images) based on the creative direction.
**Output File:** `results/media_kit.md`

**Required Content in Output:**
- Embedded generated visual assets (using correct file paths).
- Media kit details (e.g., usage instructions, recommended aspect ratios).
- Visual identity summary (recap of color palettes and typography).

**Instructions:**
When invoked, read the contents of `results/visual_direction.md` to understand the established visual themes and guidelines. Use your AI image generation tools to create specific, high-quality images corresponding to the conceptual moodboards. Compile the generated assets and the brand identity details, embedding the images correctly, into the `results/media_kit.md` file.
